# Customer Churn Prediction

This project predicts whether a telecom customer is likely to cancel ("churn") their service, based on things like their contract type, monthly charges, and how long they've been a customer.

It walks through the full process of building a machine learning model: cleaning the data, exploring it, training a few different models, comparing how well they perform, and explaining *why* the model makes the predictions it does. It also includes a simple web app where you can enter a customer's info and get a churn prediction.
